# Research Directory - db-16

This directory contains research notebooks, data source documentation, and ETL pipeline development work for the Flood Risk Assessment Database.

## Contents

- `etl_elt_pipeline.ipynb` - Main ETL/ELT pipeline notebook
- `data_resources.json` - Data source documentation (FEMA, NOAA, USGS, NASA APIs)
- `source_metadata.json` - Source metadata tracking

## Data Sources

- **FEMA**: National Flood Hazard Layer, Flood Map API, OpenFEMA
- **NOAA**: CO-OPS Derived Product API, Sea Level Rise Viewer
- **USGS**: Water Data for the Nation (7,000+ gauges)
- **NASA**: Global Flood Monitoring System, LANCE Near Real-Time Flood Products
